# Today Recommendation

Use this Skill when the user asks what to eat today or tonight.

- Read household kitchen context directly from scoped Culina data.
- Prefer expiring inventory and avoid recent repetition.
- Return a `today_recommendation` card.
- Do not create drafts or write business data.
